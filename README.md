## Requirement

- OS
  - FreeBSD
  - Linux
- [rcm](https://github.com/thoughtbot/rcm)


## Installation

```
$ git clone https://github.com/raviqqe/dotfiles ~/.dotfiles
$ rcup
$ updot
```
